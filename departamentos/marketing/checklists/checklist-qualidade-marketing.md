# ? Checklist de Qualidade  Marketing JC

Use este checklist antes de entregar qualquer material de marketing ao CEO.

---

## ?? BLOCO 1  ESTRATÉGIA (obrigatório para todos os materiais)

- [ ] **CRÍTICO:** O material posiciona JC como plataforma/tecnologia (não como consultor financeiro)?
- [ ] O objetivo do material está claro e mensurável?
- [ ] A persona/público-alvo está definida com precisão?
- [ ] Está alinhado ao propósito da JC ("democratizar o planejamento financeiro inteligente")?
- [ ] O produto correto está sendo comunicado (Método Planejar / AgroCredit / Fiado Pro)?

---

## ?? BLOCO 2  COPY E MENSAGEM

- [ ] **CRÍTICO:** O copy foca em benefícios (transformação) e não apenas em features?
- [ ] Existe um headline claro que promete um benefício específico?
- [ ] Há uma única mensagem central (sem sobrecarga de informação)?
- [ ] O CTA (chamada para ação) está explícito e usa verbo de ação?
- [ ] A linguagem está adequada para o público (simples para B2C, técnica para B2B agro)?
- [ ] Foram quebradas as principais objeções do público?
- [ ] Não há uso de termos proibidos ("investimento garantido", "rendimento certo", "consultoria financeira")?

---

## ?? BLOCO 3  VISUAL E DESIGN

- [ ] O visual segue a paleta de cores do produto?
- [ ] A tipografia é legível (tamanho mínimo 14px para corpo)?
- [ ] Existe hierarquia visual clara (o olho sabe onde ir)?
- [ ] O material funciona em mobile?
- [ ] O logo JC/produto está presente e bem aplicado?
- [ ] O contraste entre texto e fundo é adequado (acessibilidade)?
- [ ] O visual reforça a mensagem do copy (não contradiz nem distrai)?

---

## ?? BLOCO 4  CANAL E DISTRIBUIÇÃO

- [ ] O formato está correto para o canal (dimensões, duração, limite de caracteres)?
- [ ] O material foi adaptado para cada canal se for multi-canal?
- [ ] A segmentação de público está definida (para materiais pagos)?
- [ ] O pixel/rastreamento está instalado (para tráfego pago)?

---

## ?? BLOCO 5  RESULTADOS E MÉTRICAS

- [ ] Os KPIs foram definidos antes do lançamento?
- [ ] Existe forma de medir o resultado do material?
- [ ] Foi definida uma meta (ex: 100 leads, 5% de conversão)?
- [ ] Há plano de acompanhamento (quando revisar os resultados)?

---

## ?? BLOCO 6  COMPLIANCE (CRÍTICO)

- [ ] **CRÍTICO:** Nenhuma promessa de retorno financeiro específico?
- [ ] **CRÍTICO:** Nenhuma linguagem de "consultoria de investimentos"?
- [ ] Os dados de clientes/depoimentos têm autorização de uso?
- [ ] Termos de uso e LGPD estão respeitados (coleta de dados via formulário)?
- [ ] Promoções e ofertas têm regulamento (se exigido)?

---

## ?? PONTUAÇÃO

```
TODOS os CRÍTICOS ? = Material pode ser entregue
QUALQUER CRÍTICO ? = Corrigir antes de publicar
3+ Não-críticos ? = Revisar e melhorar antes de publicar
1-2 Não-críticos ? = Publicar com ajuste na próxima versão
```
