## 💡 ALEX HORMOZI — $100M OFFERS (FRAMEWORK COMPLETO)

### A Equação do Valor
```
VALOR =  (Resultado Desejado × Probabilidade Percebida de Alcançar)
         ─────────────────────────────────────────────────────────
         (Tempo até Resultado × Esforço e Sacrifício)

Para aumentar o valor:
  ↑ Resultado desejado (deixe mais claro e concreto)
  ↑ Probabilidade percebida (mais provas, garantias, depoimentos)
  ↓ Tempo até resultado (promessa mais rápida, onboarding melhor)
  ↓ Esforço e sacrifício (facilite o uso, reduza fricção)
```

### Value Stack — Como Empilhar Valor
A oferta final tem que parecer muito maior que o preço cobrado:

```
ESTRUTURA DE VALUE STACK:
  Produto principal              → valor âncora
  + Bônus 1 (complementar)       → valor adicional
  + Bônus 2 (acelera resultado)  → valor adicional
  + Bônus 3 (reduz risco/fricção)→ valor adicional
  + Garantia                     → inverte o risco
  ─────────────────────────────────────────────
  Valor total empilhado: R$ X.XXX
  Você paga: R$ XX/mês
```

### Os 4 Problemas que a Oferta Resolve (Drills)

Para cada obstáculo entre o cliente e o resultado, criar um "drill":
```
1. PROBLEMA: O cliente não sabe como começar
   DRILL: Tutorial de onboarding de 15 minutos

2. PROBLEMA: O cliente não tem tempo de usar
   DRILL: Configuração automática + relatório semanal por email

3. PROBLEMA: O cliente não acredita que vai funcionar para ele
   DRILL: Case do cliente similar + trial de 30 dias

4. PROBLEMA: O cliente tem medo de perder dinheiro
   DRILL: Garantia de devolução + suporte prioritário
```

## 🧮 PSICOLOGIA DE PREÇOS

### Ancoragem
```
ERRADO:  "Nosso plano custa R$ 197/ano"
CERTO:   "Valor total do pacote: R$ 523
          Hoje você leva por R$ 197/ano"
```

### Preço por Resultado (não por produto)
```
ERRADO:  "Acesso ao app por R$ 29,90/mês"
CERTO:   "Por menos de R$ 1/dia, você sabe para onde vai
          cada real do seu salário"
```

### Comparação de Custo de Oportunidade
```
"Quanto você perde por mês sem controle financeiro?
 A maioria dos nossos usuários descobre R$ 300-500 em gastos
 desnecessários na primeira semana. O app custa R$ 29,90."
```

### Efeito Decoy (isca) — 3 Planos
```
REGRA: O plano do meio sempre parece a escolha mais racional.

Básico  R$ 19/mês  → (faz o do meio parecer barato)
Pro     R$ 29/mês  → ← O cliente deve escolher esse
Elite   R$ 79/mês  → (faz o do meio parecer acessível)
```

## 🛡️ GARANTIAS QUE CONVERTEM

### Tipos de Garantia por Situação
```
GARANTIA INCONDICIONAL: "30 dias, devolução total, sem perguntas"
  → Usa quando o produto tem alta confiança de resultado
  → Ideal para novos clientes (remove medo de experimentar)

GARANTIA CONDICIONAL: "Se fizer X e não conseguir Y, devolvemos"
  → Usa quando quer filtrar clientes comprometidos
  → Mais credível para produtos de resultado

GARANTIA DE RESULTADO: "Se não economizar R$ 300 no primeiro mês, devolvemos"
  → Extremamente poderosa quando você tem dados reais
  → Requer confiança no produto
```

## ⚠️ REGRAS DO ESPECIALISTA EM OFERTAS

1. **Preço é percepção** — aumente o valor percebido antes de pensar em desconto
2. **Nunca descontar sem razão narrativa** — sempre justifique (lançamento, aniversário, oferta especial)
3. **A garantia remove risco, não dá desconto** — são coisas diferentes
4. **Teste sempre** — toda oferta é hipótese até ser validada com dados
5. **Compliance JC** — nenhuma oferta pode prometer retorno financeiro ou ganho de investimento
