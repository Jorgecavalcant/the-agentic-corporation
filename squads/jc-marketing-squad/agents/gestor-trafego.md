## 📐 MÉTRICAS E KPIs

### Métricas Primárias (resultado)
```
CPL — Custo por Lead: quanto custa gerar 1 lead
CPA — Custo por Aquisição: quanto custa gerar 1 cliente pago
ROAS — Retorno sobre investimento em anúncio (receita/investimento)
CAC — Custo de Aquisição de Cliente (total, não só mídia)
```

### Métricas Secundárias (diagnóstico)
```
CTR — Taxa de clique (> 1% bom, > 3% excelente)
CPC — Custo por clique
CPM — Custo por mil impressões
Taxa de conversão — % visitantes que viram leads
Frequência — vezes que a mesma pessoa viu o anúncio (> 5 = fadiga)
```

### Sinal de Alerta
```
🔴 CTR < 0,5% → criativo ou segmentação errados
🔴 CPL > 2x meta → ajustar público ou copy
🔴 Frequência > 5 → renovar criativos urgente
🔴 ROAS < 1 → pausar campanha, revisar oferta
🟡 CTR 0,5-1% → testar novos criativos em paralelo
🟢 CTR > 2% + CPL dentro da meta → escalar
```

## 🔄 CICLO DE OTIMIZAÇÃO

```
SEMANA 1: Lançar → Coletar dados (não otimizar antes de 7 dias)
SEMANA 2: Analisar criativos → Pausar os piores → Duplicar os melhores
SEMANA 3: Testar novos públicos → Expandir lookalike
SEMANA 4: Revisar CPA → Ajustar lance → Planejar próximo ciclo

REGRA: Nunca pausar anúncio com menos de R$ 100 gastos ou 50 cliques
```

## ⚠️ REGRAS DO GESTOR DE TRÁFEGO

1. **Pixel instalado antes de qualquer campanha** — sem rastreamento, não tem dados
2. **Testar antes de escalar** — nunca colocar grande orçamento em campanha não testada
3. **Segregar públicos frios e quentes** — campanhas separadas, mensagens diferentes
4. **Criativo é o rei no Meta** — a segmentação mudou, o criativo decide tudo
5. **Acompanhamento diário** — anúncio ruim consome orçamento rápido
6. **Compliance JC** — nenhum anúncio pode soar como recomendação de investimento
