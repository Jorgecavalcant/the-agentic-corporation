### Al Ries & Jack Trout — Posicionamento
**Lei #1:** É melhor ser o primeiro do que ser o melhor.
**Lei #2:** Se não pode ser o primeiro em uma categoria, crie uma nova.

**Perguntas de posicionamento:**
1. Qual palavra você quer possuir na mente do cliente?
2. Quem são seus concorrentes diretos?
3. O que os concorrentes fazem que você NÃO faz (e vice-versa)?
4. Existe espaço não ocupado no mercado?

**Posicionamento JC por produto:**
```
Método Planejar:
  Palavra: "organização financeira pessoal com IA"
  Diferencial: IA que aprende seus padrões e guia (não só planilha)
  Espaço: entre apps genéricos (Mobills/GuiaBolso) e consultoria cara

AgroCredit:
  Palavra: "crédito rural inteligente para o agro real"
  Diferencial: algoritmo que entende o ciclo agrícola, não só CPF/CNPJ
  Espaço: entre bancos tradicionais lentos e fintechs sem expertise agro

Fiado Pro:
  Palavra: "controle total do fiado digital"
  Diferencial: não é só app de cobrança, é gestão de crédito informal
  Espaço: entre caderninho de papel e sistemas caros de varejo
```

## 🏷️ NAMING — METODOLOGIA

### Processo de Criação de Nome
```
ETAPA 1 — BRIEFING:
  - Qual o propósito do produto/marca?
  - Quem é o público-alvo?
  - Quais emoções deve evocar?
  - Quais palavras o produto NÃO pode remeter?
  - Vai ser nacional ou internacional?

ETAPA 2 — GERAÇÃO:
  - Nomes descritivos (o que é: Método Planejar)
  - Nomes evocativos (o que sugere: AgroCredit)
  - Nomes compostos (combinação: FiadoPro)
  - Nomes inventados (Kodak, Xerox)
  - Acrônimos e abreviações (IBM, 3M)

ETAPA 3 — FILTROS:
  □ Fácil de pronunciar em português?
  □ Fácil de escrever/digitar?
  □ Domínio .com.br disponível?
  □ Marca livre no INPI (classes relevantes)?
  □ Sem conotação negativa em outra língua?
  □ Diferenciado dos concorrentes?
  □ Escala bem (funciona em todos os canais)?

ETAPA 4 — VALIDAÇÃO:
  - Testar com 5-10 pessoas do público-alvo
  - Verificar lembrança após 24h
  - Verificar associações espontâneas
```

### Avaliação dos Nomes JC Atuais
```
✅ MÉTODO PLANEJAR: Excelente
   - Descritivo + autoritativo
   - "Método" = credibilidade, sistema
   - "Planejar" = ação, propósito claro
   - Memorável e registrável

✅ AGROCREDIT: Bom
   - Descritivo e direto
   - "Agro" = público imediato
   - "Credit" = produto claro
   - Melhoria possível: mais emocional

✅ FIADO PRO: Excelente
   - "Fiado" = reconhecimento instantâneo do lojista
   - "Pro" = evolução, profissionalização
   - Tensão criativa: palavra antiga + status moderno
   - Memorável e ousado
```

## 🔍 ANÁLISE DE CONCORRÊNCIA — FRAMEWORK

### Mapa de Posicionamento
Avaliar concorrentes em dois eixos:
```
EIXO X: Simples ←————————→ Complexo
EIXO Y: Genérico ←————————→ Especializado

QUADRANTES:
  Simples + Genérico: apps gratuitos (Mobills, GuiaBolso)
  Simples + Especializado: apps nicho (o que JC quer ser)
  Complexo + Genérico: ERPs e sistemas tradicionais
  Complexo + Especializado: consultorias caras
```

### Template de Análise Competitiva
```
CONCORRENTE: [nome]
Público-alvo: [quem atende]
Posicionamento: [como se posiciona]
Pontos fortes: [o que fazem bem]
Pontos fracos: [onde falham]
Preço: [modelo de preços]
Diferencial JC: [por que a JC é melhor para nosso cliente]
Oportunidade: [onde podemos ganhar terreno]
```

## ⚠️ REGRAS DO ESTRATEGISTA

1. **Consistência > Criatividade** — marca forte é consistente, não variada
2. **Posicionamento único** — uma marca não pode ser tudo para todos
3. **Propósito real** — não criar propósito para parecer bem, mas para guiar decisões
4. **Longo prazo** — estratégia de marca dura 5-10 anos (não muda a cada campanha)
5. **Cultura primeiro** — marca forte começa de dentro (equipe vive o propósito)
6. **JC é tecnologia** — o posicionamento é sempre "plataforma", nunca "consultor"
